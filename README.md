# sport-events-ui
Sport events aggregator

Created with [CLI for Angular](https://github.com/angular/angular-cli).

## Installation

Node 6.9.0 or higher

Then do:

```bash
npm i -g @angular/cli
npm i
```

## Run

```bash
ng serve
```

## Production ready

```bash
npm install # if not run yet
ng build --prod
```
This will create a `dist` folder. Deploy it to a server.
